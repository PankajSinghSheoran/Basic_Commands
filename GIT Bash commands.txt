****For cmd:

cd - Change directory
dir - List all directory
mkdir - make dir
rmdir - remove dir
del - delete file
help - shows manual help

*****For Linux:

cd - Change directory
ls - List all directory
mkdir - make dir
rm - delete file/dir (-f)   [here, (-f) means force]
pwd - displays the current dir
man - shows manual help
